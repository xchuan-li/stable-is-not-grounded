# Stable Is Not Grounded

**Paper 1** of the thesis *Hierarchical Causal Evidence Framework*. Covers **Layer 1 (Stability)** and **Layer 2 (Grounding)**: certifying whether a stable-correct answer is `SC-grounded` — the licensing structure `M` is counterfactually load-bearing — rather than `SC-spurious`. Establishes the containment `Accuracy ⊋ Stability ⊋ Grounding`.

> Naming note: "Inferential Yield" now names **Paper 3 — Causal Inferential Yield (CIY)** (see `P3_Causal-Inferential-Yield`). This paper, formerly under that name, is now *Stable Is Not Grounded*.
