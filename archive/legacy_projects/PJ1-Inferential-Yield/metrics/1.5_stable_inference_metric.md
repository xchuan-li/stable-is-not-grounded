# 1.5 Stable Inference Metric Prototype

---

# Overview

This document introduces the first prototype metric for evaluating:

```text
Stable Inference
```

within the Inferential Yield / Causal Inferential Yield (CIY) framework.

The goal of this metric is not to measure only:

- benchmark correctness,
- IID accuracy,
- or task completion,

but instead to evaluate:

```text
whether inference remains stable
under valid perturbations and distribution changes
```

---

# Motivation

Traditional evaluation metrics primarily measure:

```text
final correctness
```

However, previous experiments in this project demonstrated that:

```text
high accuracy does not necessarily imply stable reasoning
```

---

## Evidence from 1.1 Shortcut Shift

The shortcut shift experiment showed:

- models may achieve high IID accuracy,
- while failing under shortcut distribution shift.

This suggests that:

```text
models may rely on spurious statistical shortcuts
rather than stable causal structure
```

---

## Evidence from 1.4 Paraphrase Stability

The paraphrase stability experiment showed:

- semantically equivalent reformulations,
- lexical abstraction,
- and structural paraphrases

can change model predictions.

This suggests that:

```text
semantic invariance
is not guaranteed
by benchmark performance alone
```

---

# Relation to 1.3 Stable Inference

This metric prototype directly builds on the framework proposed in:

```text
1.3 Stable Inference as Verified Minimal Causal Structure
```

In 1.3, stable inference is defined as:

```text
verified causal-support stability
under valid perturbations,
interventions,
and new environments
```

The current metric is only an:

```text
initial operational approximation
```

of that framework.

Because intervention-level experiments have not yet been fully developed, this prototype currently evaluates:

- behavioral stability,
- perturbation consistency,
- and distribution robustness.

---

# Core Idea

The central hypothesis is:

```text
Stable inference
should remain consistent
under semantic-preserving perturbations
and environmental changes.
```

Therefore:

```text
accuracy alone
is insufficient
for reasoning evaluation.
```

---

# Stable Inference Score (SIS)

The first prototype metric introduced here is:

```text
Stable Inference Score (SIS)
```

The initial version combines:

| Component | Meaning |
|---|---|
| IID Accuracy | original-distribution performance |
| Shift Robustness | stability under shortcut distribution shift |
| Paraphrase Consistency | stability under semantic-preserving reformulation |

---

# Initial Prototype Formula

The simplest prototype formulation is:

```text
SIS = (IID + Shift + Paraphrase) / 3
```

where:

| Variable | Meaning |
|---|---|
| IID | IID accuracy |
| Shift | distribution-shift robustness |
| Paraphrase | paraphrase consistency |

---

# Example Calculation

Using current project experiments:

| Component | Score |
|---|---|
| IID Accuracy | 1.00 |
| Shift Robustness | 0.00 |
| Paraphrase Consistency | 0.75 |

The resulting Stable Inference Score becomes:

```text
SIS = (1.00 + 0.00 + 0.75) / 3
    = 0.58
```

---

# Interpretation

A model may achieve:

```text
high benchmark accuracy
```

while still obtaining:

```text
low Stable Inference Score
```

This suggests:

```text
surface-level statistical behavior
rather than stable semantic or causal inference
```

The metric therefore attempts to separate:

```text
correctness
```

from:

```text
inference stability
```

---

# Why This Matters

Traditional evaluation often assumes:

```text
correct prediction
≈
correct reasoning
```

The CIY framework instead proposes:

```text
correctness
and
stable inference
are different properties
```

A system may:

- produce correct answers,
- while relying on unstable shortcuts,
- fragile lexical patterns,
- or distribution-specific correlations.

The current SIS formulation is intentionally simple.

It only measures behavioral-level stability and does not yet incorporate:

- perturbation difficulty,
- semantic graph invariance,
- representation drift,
- intervention consistency,
- or multi-layer reasoning structure.

Therefore, the current SIS should be interpreted as:

an early operational prototype,
rather than a finalized reasoning metric.

---

# Toward Multi-Layer Stability Evaluation

The current SIS metric is intentionally simple.

Future versions may include:

| Layer | Possible Stability Metric |
|---|---|
| Output Layer | prediction consistency |
| Distribution Layer | shift robustness |
| Semantic Layer | paraphrase invariance |
| Structural Layer | dependency / SRL stability |
| Semantic Graph Layer | AMR invariance |
| Representation Layer | latent representation drift |
| Intervention Layer | causal intervention consistency |

---

# Linguistic Structure and Stability

Future versions of Stable Inference evaluation may incorporate:

- Dependency Parsing,
- Semantic Role Labeling (SRL),
- and Abstract Meaning Representation (AMR).

These structures may help formalize:

```text
what counts as a valid semantic-preserving perturbation
```

rather than relying only on human intuition.

This is important because:

```text
surface wording may change
while semantic structure remains stable
```

---

# Current Limitations

This prototype metric has several limitations:

- only behavioral-level stability is currently measured,
- intervention-level validation is incomplete,
- semantic preservation is not yet formally verified,
- and latent representation stability is not yet included.

Therefore:

```text
SIS is currently
an early research prototype
rather than a finalized reasoning metric
```
